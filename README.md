# Turing complete game save
这是Steam游戏 [图灵完备](https://store.steampowered.com/app/1444480/Turing_Complete/) 的通关游戏存档

游戏版本0.1055Beta

## 如何使用
将文件夹`.\schematics`复制到目录`C:\Users\UserName\AppData\Roaming\Godot\app_userdata\Turing Complete\` 下然后重启游戏。
